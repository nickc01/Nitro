Nitro

A Powerup Helper for the Unity3D Game Engine

By Nicholas Cullen

Github Documentation Download Play Demo

What is it?

Nitro is a library for Unity that makes it trivial to add powerups to your game.

Whether it be for a racing game, an RPG, or a battle royal, Nitro has your back.

Creating a powerup is as simple as overriding a single class and adding it into a GameObject

See Quick Start Guide 
//Example Powerup
using Nitro;
using UnityEngine;

public class SimplePowerup : Powerup
{
                public override void DoAction()
    {
        Debug.Log("Powerup Executed!");
    }
}

Features

Nitro includes many tools and components that make it easier to integrate powerups into your game, such as:

  • Nitro Variables that make it easy to adjust variables like velocity
  • Collectors that can store and execute powerups
  • Collidables that make it easier to manage collisions

Combinable Powerups

Nitro also includes the ability to create combinable powerups, which allow multiple powerups to be combined into a single powerup with enhanced effects

For example, you could combine a fire powerup and a water powerup so that a puddle spawns with fire on it
